# 2026-07-04 低温喷雾干燥机行业情报日报

## 执行摘要

- 🔵 **已验证事实**：截至 2026-07-04，GLP-1 需求端出现新的实质性放量信号。美国 Medicare `GLP-1 Bridge` 已于 **2026-07-01** 启动，符合条件的 Medicare / Medicare Advantage 受益人可按 **50 美元/月** 获得部分品牌 GLP-1 减重药物，项目持续至 **2027-12-31**。这意味着 2026 年下半年的多肽 API、口服固体制剂、中试放大与粉体工程需求更可能从“预期”转向“交付排期”。
- 🔵 **已验证事实**：监管侧继续压缩非合规替代品空间。FDA 已在 **2025-02-21** 判定 semaglutide injection shortage resolved，并在 **2026-04-30** 提出将 `semaglutide / tirzepatide / liraglutide` 排除出 `503B bulks list`，核心含义是当 FDA 批准药物可获得时，外包配制渠道的合法空间进一步收紧，利好合规 API、受控干燥和可审计 CMC 路线。
- 🔵 **已验证事实**：`peptide CDMO` 与 `spray drying CDMO` 继续扩能。CordenPharma 于 **2026-05-27** 收购 AmbioPharm，新增美国 South Carolina 与上海的 peptide API 产能，并明确补强 `purification` 与 `lyophilization`；Symeres 于 **2026-06-10** 扩展 New Jersey Cranbury 站点的 `integrated spray drying capabilities`，将实验室级与中试级喷雾干燥纳入同一 CMC 环境。
- 🔵 **已验证事实**：益生菌与功能食品端在本周出现更强商业化场景。Vitafoods Asia 于 **2026-07-01** 明确将 `next-generation probiotics` 与 `science-backed botanicals` 作为重点；Daré Bioscience 同日宣布 Flora Sync LF5 上市，由 **Probiotical S.p.A.** 按 `cGMP` 生产，说明高活性益生菌粉体的定向应用与商业转化仍在加速。
- 🔴 **关键判断**：`35C inlet low-temperature spray dryer` 当前最强切入点仍然不是“全面替代 lyophilization / freeze drying”，而是承担 `candidate screening`、`pre-lyophilization offload`、`dry intermediate generation`、`lab-to-market scale-up` 的连续化验证工具。
- 🔴 **研发结论**：对 `GLP-1 / peptides / proteins / enzymes / probiotics`，决定成败的关键词仍然是 `outlet temperature`、`product temperature`、`residence time`、`residual moisture`、`water activity`、`particle size distribution`、`activity retention rate`，而不是单一“35C 进风”口号。

- 🔵 **spray drying CDMO 继续从单点服务转向一体化 CMC 能力**：Symeres 于 **2026-06-10** 宣布扩展 Cranbury 站点 `integrated spray drying capabilities`，支持从 `preclinical development` 到 `Phase II clinical studies`。

- 🔵 **aseptic spray drying 仍在向高门槛制药场景推进**：GEA ASEPTIC-SD 页面持续强调 `sterilization by clean steam`、真空排除空气袋与无菌粉体制备。  
  判断：高端制药客户并未放弃 spray drying，而是在把它推向更高门槛的 sterile / aseptic 场景。

## 2. 研发工艺匹配报告 (R&D & Process Alignment Report)

### 已验证事实

- 🔴 **对热敏体系，真正要控制的是产品热史而不是名义进风温度**：GEA 明确强调无菌 spray drying 的系统要求包含 `sterilization by clean steam` 和真空辅助去除 air pockets，这说明高价值体系的关键是系统级温度、无菌边界和干燥路径控制，而不是单一温度数字。

- 🔴 **传统冻干瓶颈仍真实存在于多肽链路中**：CordenPharma 本次新增的 North Augusta 产能明确聚焦 `purification and lyophilization of peptide APIs`。  
  结论：这不是“冻干将被立刻替代”的信号，反而证明 lyophilization 仍是多肽商业化的重要瓶颈节点，因此低温喷雾干燥更适合从分流与卸载切入。

- 🔴 **喷雾干燥的工业价值正在集中于连续化筛选和加速开发**：Symeres 的扩能聚焦 `ASDs`、`particle engineering`、`solubility optimisation`，并把喷雾干燥放进统一 CMC 环境。  
  结论：喷雾干燥的商业价值首先是开发速度与工艺整合，而不是单独的能耗卖点。

- 🔴 **益生菌商业化仍高度依赖定向载体、活菌稳定与常温运输能力**：Vitafoods Asia 对 `next-generation probiotics` 的展示重点，以及 Daré / Probiotical 的实际商业化路径，都在证明“菌株 + 应用场景 + 剂型稳定”是成败关键。

### 来源推断

- 🔴 **35C 进风低温喷雾干燥最适合承担四类任务**：  
  1. `candidate screening`  
  2. `pre-lyophilization offload`  
  3. `dry intermediate generation`  
  4. `lab-to-market scale-up` 的连续化验证  
  对最终要求极低残余水分、超长货架期或严格无菌终产品的体系，仍应与 `lyophilization`、`spray freeze drying` 或 `aseptic spray drying` 并行对照。

- 🔴 **GLP-1 / 多肽体系更适合从“中间体干燥 + 工艺分流”切入**：FDA 在合规性上的持续收紧，使真正有支付能力的客户更重视可审计开发路线。  
  推断：我方更现实的切入口是帮助客户减少一部分冻干批次、加快筛选节奏、缩短前中期开发周期，而不是直接宣称终局替代。

- 🔴 **功能食品 / 益生菌更容易更快形成演示案例，但单项目价值通常低于 pharma**。  
  推断：其更适合作为样板市场和现金流入口，而非当天主攻的最高价值 BD 方向。

### 我方建议的小试指标

- 🔴 **GLP-1 / peptides / proteins**
  - `outlet temperature / product temperature`：首轮建议锁在 `<= 35-40C`
  - `residence time`：优先压到 `<= 1-2 s`
  - `residual moisture`：首轮筛选建议 `1.5% - 4.0%`
  - `water activity`：首轮建议 `aw <= 0.20 - 0.30`
  - `particle size distribution`：按用途分层；口服 / 常规粉体建议 `D50 20-100 um`
  - `activity retention rate / peptide stability / protein aggregation`：以 HPLC、SEC、LC-MS 建基线，活性保留率目标先设 `>= 90%`

- 🔴 **enzymes / probiotics**
  - `probiotic viability`：首轮内部门槛建议干燥后损失 `<= 1.0-1.5 log`
  - `enzyme activity retention`：建议首轮门槛 `>= 85-90%`
  - `water activity`：优先压到 `<= 0.20`
  - `reconstitution behavior`：`30-60 s` 内可均匀分散，无明显团聚

- 🔴 **natural extracts / functional food / new materials**
  - 以 `wall material`、`feed solids`、`particle morphology`、`bulk density`、`caking tendency`、`oxidative stability` 联合评价
  - 不建议仅用含水率判断成功，必须同步看 `water activity`

- 🔵 **已验证事实**：支付端、监管端、CDMO 扩产端正在形成同向合力。

  这意味着客户更愿意为“可放大、可验证、可合规”的粉体工艺与中试验证买单。

- 🔴 **来源推断**：我方 `35C inlet low-temperature spray dryer` 在 2026 下半年最可信的市场定位仍然是：  
  `lyophilization offload tool`、`development accelerator`、`activity-retention screening platform`、`scale-up enabler`

- 🔴 **来源推断**：如果对外直接宣传“完美替代冻干”，在 pharma 客户前风险偏高；更稳妥的话术应是“先用小案例证明能否减少冻干批次、缩短周期并尽可能保住活性与复溶表现”。

- 🟢 **我方可切入机会**
  - 对 `peptide CDMO`：卖点是 `shorter cycle time + screening speed + lab-to-market scale-up`
  - 对 `biologics / proteins`：卖点是 `activity retention + continuous production + freeze-drying workload reduction`
  - 对 `probiotics / enzymes`：卖点是 `viability retention + energy saving + scalable encapsulation`
  - 对 `natural extracts / functional food`：卖点是 `continuous encapsulation + lower thermal burden + powder handling improvement`

## 5. 当日行动建议 (Action Items)

1. 🔴 **先做最小验证，不先讲宏大替代**
   选择一个 `GLP-1 model peptide` 或 `protein / enzyme model`，做 `2 x 2` 小试矩阵：  
   - 路线 A：`35C inlet low-temperature spray drying`  
   - 路线 B：`lyophilization / freeze drying` 或 `vacuum spray drying`  
   - 配方 A：基础保护剂  
   - 配方 B：增强保护剂  
   输出统一 KPI：`activity retention`、`residual moisture`、`water activity`、`particle size distribution`、`reconstitution`

2. 🔴 **把关键词先配置成对外技术语言**
   本周所有 BD 与技术资料统一围绕以下关键词：  
   `35C inlet low-temperature spray dryer`、`lyophilization alternative`、`activity retention`、`continuous production`、`lab-to-market scale-up`、`energy saving`、`outlet temperature`、`residence time`、`water activity`、`residual moisture`、`peptide stability`、`probiotic viability`

3. 🔵 **BD 优先级重新排序**
   - 第一优先级：`peptide CDMO / GLP-1 supply chain`
   - 第二优先级：`pharmaceutical spray drying CDMO / biologics formulation teams`
   - 第三优先级：`probiotics / enzymes / functional food encapsulation`
   - 第四优先级：`natural extracts / heat-sensitive new materials`

4. 🟢 **第一句话改成可成交的话术**
