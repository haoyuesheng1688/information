# 2026-07-08 低温喷雾干燥机行业情报日报

## 执行摘要

- 🔵 **已验证事实：** 截至 **2026-07-08**，GLP-1 需求侧最强新增信号仍来自 **CMS 于 2026-07-01 启动的 Medicare GLP-1 Bridge**，合资格受益人可按 **50 美元/月**获得部分 GLP-1 药物，说明减重与代谢类多肽的商业化放量逻辑继续强化。
- 🔵 **已验证事实：** **FDA** 已在 **2026-04-01** 明确提出，随着全国 GLP-1 供应趋于稳定，503A/503B compounding 的合规边界将被重新收紧；同时 **2026-07-23 至 2026-07-24** 的 PCAC 会议将继续讨论多种 peptide bulk substances，监管对“可审计、可放大、可验证”的多肽制造链要求继续上升。
- 🔵 **已验证事实：** 上游 CDMO 与制剂平台都在扩容。**CordenPharma** 于 **2026-03-04** 扩大 Boulder peptide development capacity；**ESTEVE CDMO** 于 **2026-03-20** 宣布 Girona 新增 spray drying infrastructure；**Axplora** 于 **2026-02-18** 宣布 Le Mans 扩建 lyophilization；**Genentech** 于 **2026-01-20** 将 Holly Springs 代谢类 biologics 制造投资提高至约 **20 亿美元**。
- 🔵 **已验证事实：** 功能食品与活性原料侧，**Morinaga Milk** 于 **2026-07-07** 取得 postbiotic `LAC-Living+` 的 self-affirmed GRAS，强调 postbiotic 在 heat、moisture 和 shelf-life 条件下更易配方化；**Del Monte + Treatt** 于 **2026-07-07** 推出 upcycled fruit extract range，表明天然提取物粉体化与 clean-label 原料需求继续升温。
- 🔴 **关键判断：** `35C inlet low-temperature spray dryer` 当前最稳妥的市场定位，不是对所有物料宣称“完美替代 lyophilization / freeze drying”，而是作为 **activity retention screening tool + freeze-drying offload tool + lab-to-market scale-up enabler**，优先切入多肽、蛋白、生物制剂、后生元、益生菌和天然提取物。

## 1. 全球市场动态 (Global Market Dynamics)

### 已验证事实

- 🔵 **GLP-1 支付端继续放量。** CMS 在 **2026-07-01** 启动 `Medicare GLP-1 Bridge`，把部分 GLP-1 药物可及性进一步前推到 Medicare 人群。  
  含义：2026 年下半年，多肽 API、制剂开发、无菌灌装、粉体中间体和放大验证的真实采购需求会继续增加，而不只是资本市场预期。

- 🔵 **GLP-1 灰色供应链监管持续收紧。** FDA 在 **2026-04-01** 的说明中强调，随着 national GLP-1 supply stabilized，compounders 需满足更严格条件；同时 FDA 已排定 **2026-07-23 至 2026-07-24** 的 peptide-related 503A Bulks List 讨论。  
  含义：多肽市场的客户会更重视 GMP、traceability、validated process 和更短但可审计的工艺转移路径。

- 🔵 **多肽 CDMO 正在加速补齐“开发 + 制造”一体化能力。** CordenPharma 在 **2026-03-04** 新租用 Boulder **64,000 sq. ft.** peptide development lab；其 2026 年注射剂扩建说明也明确继续增加 aseptic fill-finish 和可容纳 large-scale lyophilization 的空间。  
  含义：GLP-1、oral peptide、injectable peptide 客户采购时，越来越偏向一体化供应链，而不是只买单点设备。

- 🔵 **spray drying CDMO 供给侧继续扩建。** ESTEVE CDMO 在 **2026-03-20** 宣布 Celrà 新增 **3,000 m²** 单元，规划 `PSD-2`、`PSD-3`，并预留 commercial-scale `PSD-4`。  
  含义：pharmaceutical spray drying 正从“特殊能力”变成“平台能力”，竞争焦点会从“有没有 spray drying”转向“对热敏物料有没有低热史、低剪切、可复现的窗口”。

- 🔵 **冻干产能并未退场，反而继续投资。** Axplora 在 **2026-02-18** 宣布 Le Mans 扩建 lyophilization，且明确是响应对 speed、predictability 和 integrated handling 的需求。  
  含义：市场现实不是“喷雾干燥替代冻干”，而是“谁能帮助客户减少冻干瓶颈、缩短开发周期、并筛出适合继续冻干或转喷雾的物料”。

- 🔵 **biologics 产能在代谢类方向继续加码。** Genentech 于 **2026-01-20** 把 Holly Springs 新厂投资提高到约 **20 亿美元**，并明确瞄准 metabolic conditions。Thermo Fisher 近期也强调 Grand Island dry powder media capacity 增强，用于疫苗和 biologic therapies 的关键原材料。  
  含义：围绕 GLP-1、蛋白、mRNA 和疫苗上游原料链，干粉化、常温稳定化、快速放大验证的需求仍在升高。

- 🔵 **功能食品与天然活性原料侧，postbiotic 与 clean-label extract 在加速商用。** Morinaga Milk 的 `LAC-Living+` 于 **2026-07-07** 获得美国 self-affirmed GRAS；Del Monte 与 Treatt 同日推出 fruit-derived extract range。  
  含义：在食品和营养赛道，稳定性、可持续来源、方便配方化比“活菌必须全程存活”更容易形成商业闭环，后生元/提取物粉体化是近期更现实的突破口。

### 来源推断

- 🔴 未来 6-12 个月，`peptide CDMO + sterile injectable + powder intermediate` 的组合能力会比单点 spray dryer 更值钱。
- 🔴 对 pharma 客户而言，`shorter cycle time` 和 `freeze-dryer bottleneck relief` 的购买意愿，大概率高于单纯 `energy saving` 叙事。
- 🔴 对 food / nutraceutical 客户而言，`postbiotic stability`、`clean-label extract` 和 `continuous powder production` 会比“极低进风温度”本身更容易成交。

### 我方可切入机会

- 🟢 面向 GLP-1 / peptide 客户，切入 `candidate screening`、`dry intermediate generation`、`pre-lyophilization offload`。
- 🟢 面向 biologics / proteins 客户，切入 `activity retention`、`water activity control`、`lab-to-market scale-up`。
- 🟢 面向 postbiotic / probiotics / natural extracts 客户，切入 `stability-first powder design`、`continuous encapsulation`、`shorter development cycle`。

## 2. 研发工艺匹配报告 (R&D & Process Alignment Report)

### 已验证事实

- 🔴 **aseptic spray drying 已被 industry media 明确定位为 lyophilization alternative，而不是简单替代品。** Pharmaceutical Technology 在近期文章中指出，aseptic spray drying 对 parenteral biologic formulations 的价值在于 continuous processing、particle engineering 和避免部分 freezing stress，但 sterility assurance 和 cGMP sterile operation 仍是核心门槛。

- 🔴 **冻干产能仍是高价值工艺资源。** Axplora 和 CordenPharma 的扩建都说明，商业化客户并未抛弃 lyophilization；真正高价值的是在开发早期先判断哪些分子适合进入 spray drying route，哪些应保留 freeze drying route。

- 🔴 **postbiotic 比 live probiotic 更适合近期快速导入低温粉体化方案。** Morinaga 明确强调 `LAC-Living+` 作为 non-viable postbiotic，在 heat、moisture、extended shelf-life 环境中更稳定，这对喷雾干燥和常温配方都更友好。

### 小试关键词与验证指标

- 🔴 **GLP-1 / peptides / proteins 必测 CTQ：**
  - `outlet temperature`
  - `residence time`
  - `residual moisture`
  - `water activity`
  - `particle size distribution`
  - `activity retention rate`
  - `peptide chemical stability`
  - `protein aggregation`
  - `reconstitution behavior`

- 🔴 **enzymes / probiotics / postbiotics 必测 CTQ：**
  - `enzyme activity retention`
  - `probiotic viability` 或 `postbiotic functional marker consistency`
  - `water activity`
  - `residual moisture`
  - `particle morphology`
  - `storage stability`

- 🔴 **natural extracts / functional food / heat-sensitive materials 必测 CTQ：**
  - `encapsulation efficiency`
  - `oxidative stability`
  - `caking tendency`
  - `bulk density`
  - `flowability`
  - `reconstitution / dispersibility`

### 最优定位方案

- 🔴 **方案 A：直接宣称“35C 完美替代冻干”**
  - 优点：市场传播最直接。
  - 缺点：技术反问风险最高，且与当前 CDMO / 冻干继续扩建的事实冲突。

- 🔴 **方案 B：作为“低热史筛选 + 冻干减载 + 快速放大”工具进入**
  - 优点：与 CMS、FDA、CDMO 扩建、冻干瓶颈、postbiotic 稳定化等事实同向，最容易形成真实小试闭环。
  - 缺点：前期需要更多样品验证数据。

- 🔴 **本日报推荐最优方案：选择方案 B。**
  - 先做最小案例：
  - `GLP-1 model peptide` 或 `protein / enzyme / postbiotic model`
  - A 路线：`35C inlet low-temperature spray drying`
  - B 路线：`lyophilization / freeze drying` 或客户现有基线
  - 同时比较 `activity retention / residual moisture / water activity / reconstitution / stability`

## 3. 市场与前景评估报告 (Market & Outlook Report)

### 已验证事实

- 🔵 需求端最强主线仍是 `GLP-1 + peptide CDMO + injectable supply chain`。
- 🔵 供给端最强主线是 `spray drying capacity expansion + lyophilization capacity expansion + biologics manufacturing localization` 并行，而不是单路线替代。
- 🔵 食品与营养端最强近期信号是 `postbiotic commercialization + clean-label natural extract`，这两个方向对低温粉体化的落地速度通常快于创新药正式商业化。

### 来源推断

- 🔴 未来 3-6 个月，最有机会成交的客户不会先买“设备概念”，而会先买“验证服务包”。
- 🔴 未来 6-12 个月，市场会奖励能把 `screening -> pilot -> analytical characterization -> scale-up` 串起来的方案商。
- 🔴 对 peptide / biologics 客户，若不能给出 `water activity + residual moisture + stability` 的可验证证据，单讲 `35C inlet` 基本无法形成高质量商机。

### 我方可切入机会

- 🟢 **优先市场一：peptide CDMO / GLP-1 supply chain**
  - 诉求：减少冻干排队、提早看到粉体路线、缩短工艺开发周期。

- 🟢 **优先市场二：biologics / protein formulation teams**
  - 诉求：降低热史与冻结应力、保持活性、建立可转移的粉体中间体。

- 🟢 **优先市场三：postbiotics / probiotics / enzymes**
  - 诉求：提高货架期稳定性、控制水活度、支持常温产品开发。

- 🟢 **优先市场四：natural extracts / functional beverage ingredients**
  - 诉求：clean-label、稳定活性、可持续来源、可连续化生产。
