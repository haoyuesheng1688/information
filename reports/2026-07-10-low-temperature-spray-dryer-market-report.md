# 2026-07-10 低温喷雾干燥机行业情报日报
## 执行摘要

- 🔵 **已验证事实：** 截至 **2026-07-10**，FDA 已公布 **2026-07-23 至 2026-07-24** Pharmacy Compounding Advisory Committee 会议议程，涵盖 **MOTs-C、BPC-157、KPV、TB-500、Semax、Epitalon、Emideltide** 等多种 peptide bulk substances，说明美国多肽相关配制、审评与合规边界仍在快速收紧。
- 🔵 **已验证事实：** **Morinaga Milk** 于 **2026-07-06/07** 宣布其 postbiotic 原料 `LAC-Living+` 获得美国 self-affirmed GRAS；**Del Monte Corporation / Treatt** 于 **2026-07-07** 推出 upcycled fruit extract beverage range，说明热敏功能原料与天然提取物的粉体化、常温稳定化仍在持续放量。
- 🔵 **已验证事实：** 2026 年上半年，**CordenPharma**、**ESTEVE CDMO**、**Axplora**、**Genentech**、**Lilly**、**Novo Nordisk** 均继续围绕 peptide、injectables、spray drying、lyophilization、aseptic production 或 metabolic medicines 扩产，表明下游不会只接受“单一设备卖点”，而会优先采购可放大的工艺与供应链能力。
- 🔴 **关键判断：** `35C inlet low-temperature spray dryer` 当前最强定位不是泛化宣称“全面替代 lyophilization / freeze drying”，而是 **低热史筛选平台 + 冻干减载工具 + 中试放大加速器**。优先切入 **GLP-1 / peptide CDMO、biologics / proteins、酶制剂 / postbiotics、天然提取物 / 功能食品包埋** 四类场景。
- 🟢 **我方机会：** 对医药客户主打 `activity retention + residual moisture + water activity + reconstitution` 的小试闭环；对食品/原料客户主打 `continuous powder production + clean-label encapsulation + shelf-stability` 的快速验证包。

## 1. 全球市场动态 (Global Market Dynamics)

### 已验证事实

- 🔵 **美国多肽监管继续升温。** FDA 在 **2026-07-23/24** 的 PCAC 会议中，将讨论多种 peptide-related bulk drug substances 的 503A Bulks List 适用性，其中 **MOTs-C** 直接对应 obesity/osteoporosis 用途。  
  含义：GLP-1 / peptide 相关客户对 **工艺可追溯、质量属性可定义、放大路径可审计** 的要求会继续上升。
- 🔵 **GLP-1 / metabolic medicines 供给端仍在扩张。** **Lilly** 于 **2026-01-30** 宣布在 Pennsylvania Lehigh Valley 投资 **35 亿美元**建设 injectable medicine and device facility，用于 next-generation weight-loss therapies，包括 **retatrutide**。**Genentech** 于 **2026-01-20** 将 Holly Springs 投资上调，用于 next-generation metabolic treatments。  
  含义：减重与代谢类药物并未因 2025 年的供给修复而降温，反而在 2026 年继续向更大规模制造网络推进。
- 🔵 **上游 peptide / CDMO 能力仍在前移。** **CordenPharma** 于 **2026-03-04** 在 Boulder 扩建 **64,000 sq. ft.** peptide development lab，并与现有 large-scale peptide manufacturing 相邻；**ESTEVE CDMO** 于 **2026-03-20** 宣布在 Girona 增加 spray dryers 与 containment。  
  含义：客户越来越偏好从开发到商业化贯通的工艺平台，单点设备采购将让位于“开发验证包 + 放大路径”采购。
- 🔵 **冻干并未退出，反而继续扩产。** **Axplora** 于 **2026-02-18** 宣布扩建 Le Mans 的 lyophilization 能力，预计 **2027** 上线；**Novo Nordisk** 在其 **2025 Form 20-F** 中披露 North Carolina、Chartres、Tianjin 等地继续增加 aseptic / finished production / API capacity。  
  含义：真实市场不是“喷雾干燥替代冻干”的单向替代，而是 **谁能帮助客户降低冻干占用、前移筛选、缩短工艺迭代周期**。
- 🔵 **食品与天然原料端近 72 小时内有新增可落地信号。** **Morinaga Milk** 的 `LAC-Living+` 获得 self-affirmed GRAS；**Del Monte Corporation / Treatt** 推出 upcycled fruit extract range，强调 natural ingredient innovation、responsible sourcing 与 beverage application。  
  含义：postbiotic、天然提取物、clean-label 包埋原料比创新药更快形成商业闭环，是低温喷雾干燥近期更容易成交的方向。

### 来源推断

- 🔴 未来 3-6 个月，市场更愿意为 **“缩短开发周期”** 与 **“缓解冻干瓶颈”** 付费，而不是先为 `energy saving` 买单。
- 🔴 GLP-1 / peptide 客户的真实关注点会从“你是不是喷雾干燥”转向“你能否给出一套 `screening -> analytical characterization -> pilot scale-up` 的证据链”。
- 🔴 食品/营养客户对 `35C inlet` 的兴趣，往往要通过 **活性保持、货架期、水活度、可分散性** 才能转化为采购，而非温度口号本身。

### 我方可切入机会

- 🟢 面向 peptide / GLP-1 客户，定位为 `pre-lyophilization offload` 与 `powder intermediate feasibility`。
- 🟢 面向 biologics / proteins 客户，定位为 `low-thermal-history screening platform`。
- 🟢 面向 postbiotics / natural extracts 客户，定位为 `stability-first continuous powder solution`。

## 2. 研发工艺匹配报告 (R&D & Process Alignment Report)

### 工艺定位

- 🔴 **最优定位方案：** 采用“**先小试筛选，再决定是否进入冻干或喷雾放大**”路线，而不是先做“全面替代冻干”的大叙事。
- 🔴 **小案例优先：** 先选 1 个 `model peptide / GLP-1 analogue` 与 1 个 `postbiotic / enzyme / natural extract model` 做对照，再决定商业化主攻材料包。

### 医药物料小试必测指标

- 🔴 `outlet temperature`
- 🔴 `residence time`
- 🔴 `residual moisture`
- 🔴 `water activity`
- 🔴 `particle size distribution`
- 🔴 `activity retention rate`
- 🔴 `peptide chemical stability`
- 🔴 `protein aggregation`
- 🔴 `reconstitution behavior`

### 功能食品/活性原料小试必测指标

- 🔴 `enzyme activity retention`
- 🔴 `probiotic viability` 或 `postbiotic functional marker consistency`
- 🔴 `encapsulation efficiency`
- 🔴 `oxidative stability`
- 🔴 `bulk density`
- 🔴 `flowability`
- 🔴 `dispersibility`
- 🔴 `7-14 day accelerated stability`

### 本日研发判断

- 🔴 **GLP-1 / 多肽：** 适合把 35C 低温喷雾干燥作为 `candidate screening tool`，目标是先识别哪些分子能在较低热史下形成可复溶、可稳定保存的 powder intermediate，再决定是否继续冻干。
- 🔴 **biologics / proteins：** 需要把 `aggregation` 与 `reconstitution` 放在与 `activity retention` 同等优先级；若蛋白聚集不可控，不能因为入口温度低就直接判定路线成立。
- 🔴 **酶制剂 / postbiotics：** 相比 live probiotics，postbiotic 对温度与货架期更友好，更适合作为近期验证入口。
- 🔴 **天然提取物 / 新材料：** 应优先关注 `caking tendency`、氧化稳定性与连续生产适配，而不是只看单次产率。

## 3. 市场与前景评估报告 (Market & Outlook Report)

### 已验证事实

- 🔵 **监管和扩产同时存在。** FDA 对 peptide compounding 的审视在变严，但 Lilly、Genentech、Novo Nordisk、CordenPharma 等仍在继续扩产，说明需求并没有降温，而是对合规与制造质量的要求更高。
- 🔵 **spray drying 与 lyophilization 将长期并存。** ESTEVE 在建 spray dryers，Axplora 在扩冻干，Novo Nordisk 在加 aseptic 与 finished production，产业链真实态势是多路线并行。
- 🔵 **功能原料端进入新一轮“稳定化优先”窗口。** Morinaga 的 postbiotic GRAS 与 Del Monte/Treatt 的天然提取物新品，都更强调 formulation flexibility、ingredient transparency、responsible sourcing 与应用开发速度。

### 来源推断

- 🔴 未来 6-12 个月，最容易签单的不是“整机替换冻干线”，而是 **验证服务包 + 小试设备 + 放大方法学** 的组合。
- 🔴 若我方能证明 `35C inlet low-temperature spray dryer` 在 **peptide / protein / enzyme / postbiotic** 上形成稳定的 CTQ 数据积累，其商业价值将高于单次设备成交。
- 🔴 `pharmaceutical spray drying CDMO` 市场正在向平台化发展，因此我方需要避免只卖“喷雾”概念，必须同时卖 **分析、验证、放大与应用边界**。

### 我方可切入机会

- 🟢 **市场一：** peptide CDMO / GLP-1 供应链  
  切入词：`freeze-dryer bottleneck relief`、`rapid formulation screening`、`powder intermediate generation`
- 🟢 **市场二：** biologics / protein formulation 团队  
  切入词：`activity retention`、`water activity control`、`low thermal history`
- 🟢 **市场三：** enzymes / postbiotics / probiotics  
  切入词：`shelf-stable powder`、`continuous production`、`faster formulation iteration`
- 🟢 **市场四：** natural extracts / functional food ingredients / heat-sensitive materials  
  切入词：`clean-label encapsulation`、`responsible sourcing`、`pilot-scale powdering`
